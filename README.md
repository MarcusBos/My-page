# Portfolio Website

A small multi-page personal portfolio built with HTML, CSS, JavaScript, and Bootstrap Collapse for navigation.

## Mobile-First Adaptation

This project is adapted for phones and small screens:
- Responsive breakpoints at `992px` and `640px`
- Scaled navigation menu and typography for mobile view
- Repositioned content blocks for better readability on handheld devices
- `viewport` meta tag enabled for proper rendering on smartphones

## Quick Start

1. Open a terminal in this project folder.
2. Run:

```bash
python3 -m http.server 8000
```

3. Launch in browser:

[http://localhost:8000/index.html](http://localhost:8000/index.html)
